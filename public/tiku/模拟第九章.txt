一、单选题（共30题）
1. 微信小程序云开发中，初始化云开发环境的 API 是？
A. wx.cloud.init()
B. wx.initCloud()
C. cloud.init()
D. wx.cloud.start()
答案：A
解析： 小程序端使用 wx.cloud.init({ env: '环境ID' }) 初始化云开发环境，需在 app.js 的 onLaunch 中调用。cloud.init() 是服务端（云函数）使用的 API。
2. 云开发数据库中，创建一个集合（Collection）相当于关系型数据库中的什么概念？
A. 创建一条记录
B. 创建一张表
C. 创建一个数据库
D. 创建一个索引
答案：B
解析： 云开发数据库是 MongoDB 风格的文档型数据库，集合（Collection）对应关系型数据库中的"表"，文档（Document）对应"记录/行"。
3. 云开发数据库集合的默认权限"所有人可读写"适用于什么场景？
A. 生产环境正式业务
B. 仅开发调试阶段
C. 金融支付类应用
D. 用户隐私数据存储
答案：B
解析： "所有人可读写"是默认权限，仅适合开发调试。生产环境应根据业务设置为"仅创建者可读写"或自定义安全规则，防止数据被恶意篡改。
4. 在云函数中获取当前调用用户的 OpenID，正确的方式是？
A. wx.getWXContext()
B. cloud.getWXContext().OPENID
C. wx.cloud.getWXContext()
D. cloud.getWXContext().openId
答案：B
解析： 云函数中使用 cloud.getWXContext() 返回上下文对象，包含 OPENID（大写）、APPID、UNIONID 等属性。注意属性名是大写的 OPENID。
5. 小程序端上传文件到云存储，应使用哪个 API？
A. wx.uploadFile()
B. cloud.uploadFile()
C. wx.cloud.uploadFile()
D. wx.cloud.storage.uploadFile()
答案：C
解析： 小程序端使用 wx.cloud.uploadFile()，参数包括 cloudPath（云存储路径）和 filePath（本地临时文件路径）。wx.uploadFile() 是上传到普通服务器的 API。
6. 云函数定时触发器的配置文件中，触发周期使用什么格式？
A. Unix 时间戳
B. Cron 表达式
C. ISO 8601 格式
D. 毫秒数
答案：B
解析： 云函数定时触发器在 config.json 中使用 Cron 表达式配置触发周期，如 "0 0 2 1 * * *" 表示每月1日凌晨2点执行。
7. 云开发-云调用（Cloud Call）相比传统 HTTPS 调用的主要优势是？
A. 传输速度更快
B. 免鉴权调用微信服务端接口
C. 支持更大文件传输
D. 可以跨域访问
答案：B
解析： 云调用是云开发特有的能力，可以在云函数中直接调用微信服务端接口（如发送模板消息、获取小程序码），无需手动处理 access_token 等鉴权逻辑。
8. 在 TODO 实战项目中，使用云开发数据库实现数据增删改查时，添加数据使用的方法是？
A. db.collection('todos').add()
B. db.collection('todos').insert()
C. db.collection('todos').create()
D. db.collection('todos').push()
答案：A
解析： 云开发数据库添加记录使用 .add({ data: {...} }) 方法。insert()、create()、push() 都不是云开发数据库的合法 API。
9. 云函数中操作数据库，需要引入的模块是？
A. const db = wx.cloud.database()
B. const cloud = require('wx-server-sdk')
C. const db = require('mongodb')
D. const cloud = wx.cloud
答案：B
解析： 云函数中通过 require('wx-server-sdk') 引入云开发服务端 SDK，然后初始化 cloud.init()，再通过 cloud.database() 获取数据库实例。不能使用小程序端的 wx.cloud。
10. 云开发数据库中，查询条件"年龄等于18或等于20"的正确写法是？
A. { age: { $eq: [18, 20] } }
B. { age: { $in: [18, 20] } }
C. { age: { $or: [18, 20] } }
D. { age: [18, 20] }
答案：B
解析： $in 用于匹配字段值在指定数组中的文档。$or 是逻辑操作符，用于组合多个条件，正确写法应为 { $or: [{ age: 18 }, { age: 20 }] }。

A. cloud.openapi.subscribeMessage.send()

C. wx.requestSubscribeMessage()
D. cloud.sendMessage()
答案：A
解析： 云函数中通过 cloud.openapi.subscribeMessage.send() 发送订阅消息，属于云调用能力。wx.requestSubscribeMessage() 是小程序端请求用户授权订阅的 API。

A. doc(id).delete()
B. doc(id).remove()
C. where({...}).delete()
D. doc(id).destroy()
答案：B
解析： 云开发数据库删除单条记录使用 doc(id).remove()，删除多条使用 where({...}).remove()。没有 delete() 或 destroy() 方法。

15. 云开发数据库中，对查询结果按创建时间倒序排列，正确的写法是？
A. .orderBy('createTime', 'desc')
B. .sort({ createTime: -1 })
C. .orderBy('createTime', 'descending')
D. .order({ createTime: 'desc' })
答案：A

答案：A、B、C

A. String
B. Number

D. GeoPoint
答案：A、B、C、D

A. 仅创建者可读写
B. 所有人可读写
C. 所有人可读，仅创建者可写
D. 自定义安全规则
答案：A、B、C、D

4. 云函数相比传统后端开发的优势包括？（多选）

解析： 云函数支持 async/await、Promise、callback 等异步处理方式。D 错误：Node.js 中应避免同步阻塞操作，会影响性能和并发能力。
19. 云开发数据库文档的 _id 字段特点包括？（多选）
A. 系统自动生成
B. 全局唯一
C. 可以自定义
D. 不可修改
答案：A、B、C
解析： _id 是文档唯一标识，默认系统自动生成（类似 ObjectId），也可以在 add() 时自定义。D 错误：_id 一旦创建不可修改，但可以通过删除重建间接"修改"。
20. 提升云开发数据库查询性能的方法包括？（多选）
A. 为常用查询字段创建索引
B. 使用聚合查询代替多次查询
C. 限制返回字段数量（projection）
D. 增加数据库集合数量
答案：A、B、C
解析： 优化数据库性能：创建索引、合理使用聚合、限制返回字段。D 错误：增加集合数量不会提升性能，反而增加管理复杂度，应合理设计集合结构。
