一、单选题（共30题）
1. 小程序将数据同步存储到本地缓存使用的 API 是？
A. wx.setStorage()
B. wx.setStorageSync()
C. wx.saveData()
D. wx.localStorage()
答案：B
解析： wx.setStorageSync() 是同步存储 API，会阻塞当前线程直到存储完成，适合存储少量关键数据。wx.setStorage() 是异步版本，不会阻塞。wx.saveData() 和 wx.localStorage() 不是小程序 API。注意：单个 key 允许存储的最大数据长度为 1MB，所有数据存储上限为 10MB。
2. 获取用户当前地理位置使用的 API 是？
A. wx.getLocation()
B. wx.chooseLocation()
C. wx.openLocation()
D. wx.getPosition()
答案：A
解析： wx.getLocation() 获取用户当前的地理位置（经纬度、速度、准确度等），需要用户授权。wx.chooseLocation() 打开地图选择位置（让用户手动选点），wx.openLocation() 使用微信内置地图查看位置。wx.getPosition() 不存在。
3. 小程序登录流程中，获取临时登录凭证 code 的 API 是？
A. wx.getUserInfo()
B. wx.login()
C. wx.getCode()
D. wx.authLogin()
答案：B
解析： wx.login() 获取临时登录凭证 code（有效期 5 分钟），开发者需要将 code 发送到后端，后端再用 code 换取 openid 和 session_key。wx.getUserInfo() 获取用户信息（已废弃，现用 wx.getUserProfile）。wx.getCode() 和 wx.authLogin() 不存在。
4. 本地缓存中，同步读取数据的 API 是？
A. wx.getStorage()
B. wx.getStorageSync()
C. wx.readStorage()
D. wx.loadStorage()
答案：B
解析： wx.getStorageSync('key') 同步读取本地缓存数据，如果 key 不存在会返回空字符串或抛出异常（需要用 try-catch）。wx.getStorage() 是异步版本。wx.readStorage() 和 wx.loadStorage() 不是小程序 API。
5. 使用微信内置地图查看指定位置的 API 是？
A. wx.getLocation()
B. wx.chooseLocation()
C. wx.openLocation()
D. wx.viewMap()
答案：C
解析： wx.openLocation({ latitude, longitude, name, address }) 打开微信内置地图查看指定位置，可以显示位置名称和地址。wx.getLocation() 是获取当前位置，wx.chooseLocation() 是让用户选择位置。wx.viewMap() 不存在。
6. 小程序发起微信支付调起客户端支付使用的 API 是？
A. wx.requestPayment()
B. wx.pay()
C. wx.weixinPay()
D. wx.unifiedOrder()
答案：A
解析： wx.requestPayment() 调起微信支付客户端，参数包括 timeStamp、nonceStr、package、signType、paySign 等，这些参数需要由后端调用微信统一下单接口（unifiedOrder）生成后返回给前端。wx.pay()、wx.weixinPay() 不存在，wx.unifiedOrder() 是后端接口不是前端 API。
7. 获取本地缓存信息（包括当前已用空间大小）使用的 API 是？
A. wx.getStorageInfo()
B. wx.getStorageStats()
C. wx.checkStorage()
D. wx.storageInfo()
答案：A
解析： wx.getStorageInfo() / wx.getStorageInfoSync() 获取当前本地缓存的相关信息，返回 keys（所有 key 数组）、currentSize（当前已用空间，单位 KB）、limitSize（空间限制，单位 KB）。其他选项不是合法 API。
8. 小程序中，清除所有本地缓存数据使用的 API 是？
A. wx.removeStorage()
B. wx.clearStorage()
C. wx.deleteStorage()
D. wx.cleanStorage()
答案：B
解析： wx.clearStorage() / wx.clearStorageSync() 异步/同步清除所有本地缓存数据。慎用，会清除所有数据。wx.removeStorage('key') 只删除指定 key 的数据。wx.deleteStorage() 和 wx.cleanStorage() 不存在。
9. 小程序获取用户授权设置信息的 API 是？
A. wx.getUserAuth()
B. wx.getSetting()
C. wx.checkAuth()
D. wx.authStatus()
答案：B
解析： wx.getSetting() 获取用户的当前设置，返回 authSetting 对象，包含各权限的授权状态（如 scope.userInfo、scope.userLocation 等是 true/false）。配合 wx.authorize() 提前向用户发起授权请求。其他选项不是合法 API。
10. 小程序选择本地文件使用的 API 是？
A. wx.chooseFile()
B. wx.selectFile()
C. wx.pickFile()
D. wx.openFile()
答案：A
解析： wx.chooseFile() 从客户端会话或本地选择文件，支持选择文件类型和数量。wx.selectFile()、wx.pickFile()、wx.openFile() 不存在。注意：基础库 2.10.0 开始支持，低版本需兼容处理。
11. 小程序保存文件到本地使用的 API 是？
A. wx.saveFile()
B. wx.downloadFile()
C. wx.storeFile()
D. wx.writeFile()
答案：A
解析： wx.saveFile() 保存临时文件到本地，返回保存后的文件路径（本地用户目录）。wx.downloadFile() 下载文件到临时路径，需要再调用 saveFile() 持久化保存。wx.storeFile() 不存在，wx.writeFile() 是文件系统 API（wx.getFileSystemManager）用于写入文件内容。

D. wx.showFile()

A. wx.getSetting()
B. wx.authorize()

10. 小程序本地缓存的使用注意事项包括？（多选）

B. 敏感数据建议加密存储

D. 缓存数据永久有效不会丢失
答案：A、B、C
解析： 本地缓存注意事项：① 总上限 10MB，单个 key 最大 1MB；② 敏感信息（如 token）建议加密后存储；③ 同步 API（Sync）会阻塞当前线程，大量操作建议用异步。D 错误：缓存数据在用户清理微信缓存、删除小程序、手机恢复出厂等情况下会丢失，重要数据需要服务端备份。

解析： 微信小程序云开发（Tencent Cloud Base）提供四大基础能力：云数据库（JSON 数据库，支持增删改查）、云存储（文件存储，支持上传下载）、云函数（在云端运行的代码，无需搭建服务器）、云调用（调用微信开放接口的能力，如发送订阅消息）。
15. 小程序中可用于选择媒体文件的 API 包括？（多选）

B. wx.chooseVideo()

B. 页面内 onShareTimeline()

17. 小程序中 tabBar 相关的配置和 API 包括？（多选）

B. wx.switchTab()

19. 小程序中可用于录音和音频播放的 API 包括？（多选）
A. wx.startRecord()
B. wx.getRecorderManager()
C. wx.createInnerAudioContext()

答案：B、C

20. 小程序中可用于数据缓存的策略包括？（多选）
A. 使用 wx.setStorage 系列 API 本地缓存
B. 使用全局变量 getApp().globalData
C. 使用页面间参数传递
D. 使用 MobX/Vuex 等状态管理库
答案：A、B、C、D
解析： 小程序数据缓存/状态管理策略：① wx.setStorage 系列持久化存储到本地；② getApp().globalData 全局变量，页面间共享数据但小程序退出后丢失；③ 页面跳转时通过 url 参数或 eventChannel 传递数据；④ 可引入 MobX、Vant Weapp 的 Store 等第三方状态管理方案实现更复杂的状态管理。
